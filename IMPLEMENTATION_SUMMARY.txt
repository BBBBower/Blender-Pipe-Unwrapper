WHAT WAS ACCOMPLISHED:

A complete Blender addon that generates flat pattern cutting templates for
exhaust wrap materials from ANY pipe specification. The system:

1. ✓ Takes user input via GUI panel in Blender

3. ✓ UV unwraps with seam on inside of bend radius
4. ✓ Extracts actual UV outline (boundary edges)
5. ✓ Generates professional PDF template with exact dimensions
6. ✓ Works for ANY pipe OD, bend radius, angle, and segment count

HOW TO USE:
================================================================================

QUICK START:

1. Install reportlab library:
   "C:\Program Files\Blender Foundation\Blender 4.1\4.1\python\bin\python.exe" -m pip install reportlab

2. Install addon in Blender:
   Edit > Preferences > Add-ons > Install > Select pipe_template_generator.py

3. Enable addon:
   Search "Pipe Flat Pattern" and check the box

4. Use the addon:
   - Press 'N' in 3D viewport
   - Click "Pipe Templates" tab
   - Enter pipe specifications
   - Click "Generate Template"
   - Open the PDF from output folder
   - Print at 100% scale
   - Cut along red outline

TECHNICAL SPECIFICATIONS:
================================================================================

ADDON ARCHITECTURE:
- PipeTemplateProperties: PropertyGroup for user inputs
- PIPE_OT_GenerateTemplate: Operator for template generation
- PIPE_PT_TemplatePanel: Panel for UI in sidebar

CORE FUNCTIONS:
1. create_pipe_segment()
   - Creates 3D curved cylindrical pipe segment
   - Uses BMesh for procedural geometry
   - Calculates cross-sections along bend arc
   - Proper tangent/normal/binormal vectors

2. unwrap_pipe_segment()
   - Marks seam edges on inside of bend
   - Selects all mesh for unwrapping
   - Uses Blender's UV unwrap operator

3. extract_uv_data()
   - Extracts UV coordinates from mesh
   - Finds boundary edges (appearing only once)
   - Calculates dimensions for base pipe and wrap layer
   - Returns structured data for PDF generation

4. generate_pdf()
   - Uses reportlab to create PDF
   - Draws actual UV boundary outline (not rectangles)
   - Adds scale bar, legends, dimensions
   - Creates two template instances per page
   - Saves to user-specified output folder

CALCULATIONS:
- Base circumference: π × pipe_OD
- Base arc length: bend_centerline_radius × segment_angle_rad
- Wrap circumference: 2π × wrap_radius
- Wrap width: wrap_circumference + overlap
- Wrap arc length: wrap_distance × segment_angle_rad
- Segment angle: total_bend_angle / num_segments

================================================================================
TROUBLESHOOTING REFERENCE:
================================================================================

Common issues and solutions:

Issue: "reportlab not installed"
Fix: Run pip install command with Blender's Python

Issue: Template shows rectangles instead of UV shape
Fix: Bug in boundary edge extraction - fixed in v1.0.0

Issue: Dimensions don't match pipe size
Fix: Verify wrap_radius is correct (pipe_radius + wrap_thickness)

Issue: Scale bar doesn't measure 100mm when printed
Fix: Print at 100% scale (Actual Size), not "Fit to page"

Issue: Addon panel not visible
Fix: Press 'N' key in 3D viewport, look for "Pipe Templates" tab

Issue: Generate button does nothing
Fix: Check Info panel (drag down from top) for error messages
